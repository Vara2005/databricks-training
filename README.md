# Databricks Training Repository

## Introduction
This repository contains my learning journey in SQL, Databases, and Data Engineering as part of the Databricks training program.

I am maintaining this repository to track my daily progress, practice SQL concepts, and organize my learning in a structured way.

## Objectives
The main objectives of this training are:

- Learn SQL from fundamentals to advanced concepts
- Understand relational database design
- Practice writing SQL queries
- Build consistency in daily learning
- Maintain professional project documentation

## Repository Structure
This repository is organized in a week-wise and day-wise format.

Each day contains:

- SQL scripts
- Practice queries
- Query outputs
- Learning notes
- README documentation

## Tools Used

- DB Fiddle for SQL practice
- SQL for database querying
- Structured documentation using Markdown

## Learning Goals
