# Week 1 - SQL Fundamentals

## Overview
This week focuses on learning the fundamentals of SQL and understanding how relational databases work. The goal is to build a strong foundation in database concepts and query writing.

## Topics Covered
- Introduction to Databases
- Understanding Tables, Rows, and Columns
- Primary Keys and Foreign Keys
- Creating tables using SQL
- Inserting data into tables
- Writing basic SELECT queries
- Filtering data using WHERE clause
- Sorting data using ORDER BY
- Pattern matching using LIKE operator

## Tools Used
- GitHub for version control and maintaining daily progress
- DB Fiddle for executing and testing SQL queries
- SQL scripts for schema creation and query practice

## Learning Outcome
By the end of this week, I will be able to:
- Create database tables
- Insert and manage data
- Write and execute basic SQL queries
- Understand relationships between tables
- Organize daily learning in GitHub repository

## Folder Structure
This week contains daily practice files, SQL scripts, query outputs, and learning notes.
